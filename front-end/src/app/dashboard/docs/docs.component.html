How to use?

  • By default AppModule is available as a root module.
  • Select App module (or any other node in graph) and click on "New" menu item.
  • In "New Item" dialog box, select item type and provide details. Once done, click on "Create".
  • After you have defined application structure, enter new application name in textbox and click on "Generate App" button which shall download new Angular application zip file.
  • Unzip the zip file and run "npm install" command inside directory.
  • Install Angular CLI
  • Run "ng serve --open" command to run Angular application in browser!
